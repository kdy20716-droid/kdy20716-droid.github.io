사용자에게 즐거운 경험을 주는 인터랙티브한 웹 게임 구현과 로직 설계에 깊은 관심을 가지고 있습니다.

## 🏗️ 주요 프로젝트

* **🍉 수박 게임 (Suika Game Style)**: 과일을 합쳐 점수를 내는 물리 엔진 기반의 웹 게임
* **🔫 라이어 루렛 (Liar's Roulette Style)**: 심리적 긴장감을 주는 확률 기반의 카드 루렛 게임
* **🎱 당구 게임 (Billiards Game)**: 공의 충돌과 반사 로직을 구현한 물리 시뮬레이션 게임
* **🎲 라이어 바 스타일 프로젝트**: 현재 개발 중인 심리 기반 보드 게임
